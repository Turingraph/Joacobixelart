# Folder structure of `src/`

1.	`asset/`
*	Purpose: Store image files for button logo (a.k.a. button icon).
2.	`atom/`
*	Purpose: Store reusable `.tsx` files that do not import any external file.
3.	`molecule/`
*	Purpose: Store reusable `.tsx` files that import `atom/` files.
4.	`panel/`
*	Purpose: Store non-reuseable frontend UI files.
